Circuit Virtual Private Clinic Demo

This demo shows virtual clinic where patients consult doctors via a browser. Patients get their appointments via email with a link to enter the waiting room. The clinic administrator (or doctors themselves) can view the waiting list and connect the patient with the doctor.

Source code of this demo is available on github.


How to use the demo

Login as doctor to the regular Circuit client at https://circuitsandbox.net as doctor.nick@mailinator.com with password Nick.123!

On another browser tab view the waiting list.

On another browser tab open a patient: Patient 100, Patient 101, Patient 102, Patient 103

Then connect a patient to his/her doctor using 'connect' button on the waiting list.

A new conversation is automatically created for this consultation. Start the call on that new conversation using the doctors Circuit client.
